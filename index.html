Projects

Thumb Movement Analysis Using Cadaveric Specimens

Objective: To quantify thumb-tip trajectories produced by thumb muscles, contributing to the understanding of musculoskeletal biomechanics.

Methods: Developed a motion tracking system using MATLAB, built a custom arm fixation device, and analyzed thumb movements to understand the functional contribution of muscles.

Results: Provided data for refining biomechanical models of thumb mechanics. View Project Poster

EEG Data Analysis Project

Objective: To analyze EEG data for extracting frequency bands and understanding brain activity patterns between infants and adults.

Methods: Developed MATLAB scripts to calculate band powers and applied machine learning for neural pattern recognition.

Results: Demonstrated distinct neural patterns differentiating age groups, contributing to understanding developmental neuroscience.

Code Repository: GitHub - EEG Analysis Project

Molecular Dynamic Simulation of Protein Tyrosine Phosphatase

Objective: To analyze atomic interactions through computer simulations of protein-ligand complexes.

Methods: Used Gaussian for ligand modeling and conducted molecular dynamics simulations.

Results: Provided insights into protein stability under various modifications. View Project Poster
